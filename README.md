# Notes App

A simple and interactive Notes App built using HTML, CSS, and JavaScript.

## Features
- Add notes
- Edit notes
- Delete notes
- Search notes
- Data persistence using localStorage

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project
2. Open index.html in any browser
